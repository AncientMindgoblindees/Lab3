# Black & Gold Circuit Website

This is a static website featuring a black and gold theme, a landing page with a circuit diagram, interactive battery charge meter, and a profile selection page.

## Structure
- `index.html`: Landing page with SVG circuit (battery, switch, resistor, LED) and animated battery bar (green to red as it drains)
- `profiles.html`: 2x2 grid of profile cards with image slots
- `styles.css`: Black and gold themed stylesheet, including all layout and color customizations
- `script.js`: Handles circuit animation, battery bar, and navigation

## Features
- Modern black and gold color scheme
- Interactive SVG circuit with a switch and animated LED
- Battery charge meter drains from green (100%) to red (0%) before navigating
- Profile selection page with image slots in a 2x2 grid

## Usage
Open `index.html` in your browser to view and interact with the website. Press the switch to see the battery drain and proceed to profile selection.
